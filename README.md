# 量化投资总控中心 (Quantitative Investment Control Center)

## 📌 项目简介

量化投资总控中心是一个基于 Coze 多智能体架构的**全自动量化投资系统**。系统由 1 个总控中心和 9 个专业金融 AI 专家组成，覆盖从信息采集、数据分析、策略推演、风险评估、仓位管理、交易执行到事后复盘的全闭环流程。

## 🏗️ 系统架构

```
信息专家 → 数据专家 → 信息分析专家 → 基金股票专家 → 评估专家
                                                          ↓
公司通知助手 ← 复盘专家 ← 风控止损专家 ← 执行交易专家 ← 仓位管理专家
```

## 🤖 智能体列表

| 序号 | 智能体 | 职责 |
|------|--------|------|
| 🧠 | **量化投资总控中心** | 协调全部 9 大金融 AI 专家，完成全自动闭环 |
| 📡 | **信息专家** | 24小时监控全球财经媒体，实时抓取原始信息 |
| 🔬 | **数据专家** | 数据清洗、提纯与重构，剔除噪音 |
| 📊 | **信息分析专家** | 宏观策略与基本面分析，判断利好/利空 |
| 💰 | **基金股票专家** | 深度解读数据，推荐具体标的 |
| ⚠️ | **评估专家** | 压力测试与风控审核，一票否决权 |
| 📐 | **仓位管理专家** | 凯利公式资金分配 |
| 🎯 | **执行交易专家** | 市价单/限价单执行 |
| 🛡️ | **风控止损专家** | 盘中动态盯盘，硬性止损止盈 |
| 📋 | **复盘专家** | 收盘后逆向追踪，优化专家逻辑 |
| 📢 | **公司通知助手** | 飞书群发布工作通知 |

## ⚙️ 技术栈

- **平台：** Coze (扣子) 多智能体编排
- **模型：** GLM-4.7
- **技能：** 联网搜索 (BochaWebSearch / search_web)
- **通知：** 飞书群机器人

## 🔄 工作流程

1. **信息采集** — 信息专家 24h 监控全球财经媒体
2. **数据清洗** — 数据专家提纯重构，输出 JSON 格式
3. **分析推演** — 信息分析专家 + 基金股票专家联合分析
4. **风险评估** — 评估专家全方位压力测试
5. **仓位分配** — 仓位管理专家按凯利公式分配资金
6. **交易执行** — 执行交易专家模拟实盘操作
7. **风控止损** — 风控止损专家实时盯盘
8. **复盘优化** — 复盘专家收盘后逆向追踪

## 🚀 快速开始

1. 在 Coze 平台导入本项目配置
2. 配置各智能体的 API Key 和技能
3. 配置飞书通知机器人
4. 启动总控中心
