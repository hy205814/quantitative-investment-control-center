# 量化投资总控中心 (Quantitative Investment Control Center)

> 基于 Coze 平台构建的 9 大金融 AI 专家协作系统，实现从信息抓取到交易复盘的全自动闭环。

## 📋 项目概览

本系统由 **1 个总控协调器 + 9 个金融 AI 专家** 组成，覆盖投资决策全链路：

```
信息采集 → 数据清洗 → 宏观分析 → 风控审核 → 仓位管理 → 交易执行 → 动态盯盘 → 复盘优化
```

## 🤖 AI 专家矩阵

| # | 智能体 | 角色 | 职责 |
|---|--------|------|------|
| 1 | **信息专家** | 金融与宏观资讯极速追踪器 | 24小时监控全球财经媒体、社交平台、政府公告及行业研报 |
| 2 | **信息分析专家** | 金融信息提纯与重构大师 | 去重去噪，将杂乱信息提炼为结构化 JSON 数据 |
| 3 | **数据专家** | 数据清理与事实核查 | 提供彻底的事实列表，剔除不实信息 |
| 4 | **基金股票专家** | 宏观策略与行业基本面分析师 | 深度解读事件，判断利好/利空方向及逻辑支撑 |
| 5 | **风控止损专家** | 严苛的投资风控审核员 | 从流动性、追高、监管、黑天鹅等维度全方位压力测试 |
| 6 | **仓位管理专家** | 资金分配与博弈精算师 | 基于凯利公式/风险暴露模型进行资金分配 |
| 7 | **执行交易专家** | 冷酷无情的交易指令发送器 | 模拟实盘操作，市价单/限价单建仓或清仓 |
| 8 | **评估专家** | 盘中动态盯盘与纪律执行官 | 实时监控价格波动，捍卫交易纪律，发出止损止盈指令 |
| 9 | **复盘专家** | 系统自进化引擎与首席审查官 | 每日对比预测与实际涨跌幅，逆向追踪错误根源 |

### 辅助机器人

|---|--------|------|

## 🔄 数据流

1. **信息专家** 从全球财经来源抓取实时资讯
2. → **信息分析专家** 清洗并结构化数据
3. → **数据专家** 核查事实，输出可靠数据集
4. → **基金股票专家** 进行宏观与行业分析
5. → **风控止损专家** 对候选标的进行压力测试
6. → **仓位管理专家** 按凯利公式分配资金
7. → **执行交易专家** 模拟实盘建仓/清仓
8. → **评估专家** 盘中进行实时监控与纪律执行
9. → **复盘专家** 收盘后对比分析，优化系统提示词

## 🚀 使用说明

### 环境要求

- Coze (扣子) 平台账号
- 飞书 (Feishu/Lark) 账号（用于通知推送）

### 部署步骤

1. 登录 [Coze.cn](https://www.coze.cn)
2. 导入本仓库中各 bot 的配置文件
3. 配置飞书 Webhook 通知
4. 启动并测试全链路
