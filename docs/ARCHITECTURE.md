## 三层架构详解

### Layer 1: 信息采集层 (Information Collection)

负责从多个渠道获取原始金融数据并进行处理：

- **信息专家** → 原始数据采集，24小时不间断监控
- **信息分析专家** → 数据清洗、去重、格式化
- **数据专家** → 事实核查，输出高可靠性数据集

### Layer 2: 分析决策层 (Analysis & Decision)

基于清洗后的数据进行分析和风控：

- **基金股票专家** → 宏观策略与行业基本面分析
- **风控止损专家** → 风险压力测试与评级
- **仓位管理专家** → 资金分配优化（凯利公式）

### Layer 3: 执行反馈层 (Execution & Feedback)

执行交易指令并进行复盘优化：

- **执行交易专家** → 模拟实盘操作
- **评估专家** → 盘中实时监控与纪律执行
- **复盘专家** → 收盘后复盘与系统优化
